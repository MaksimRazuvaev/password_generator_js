# Password Generator Starter Code

WebPage sugests the user to generate safe password by using custom set of lowercase, uppercase, numeric, and/or special characters.

## Description

- webPage whritten on HTML (index.html), structured with CSS (style.css), and password generator functionality is written with JavaScript (script.js)
- WebPage has next appearence:
![Initial_WebPage](/Develop/images/Initial_WebPage.PNG)
- The main section is "Generate Password" button to initiate password generation, and "Your secure Password" textarea to display generated password.
- User is taken to the set of promt messages to set up password's options:
- Password Length Prompt
![Initial_WebPage](/Develop/images/Password_Length_Prompt.PNG)
- Lowercase Prompt
![Initial_WebPage](/Develop/images/LowerCase_Prompt.PNG)
- Uppercase Prompt
![Initial_WebPage](/Develop/images/UpperCase_Prompt.PNG)
- Numeric Prompt
![Initial_WebPage](/Develop/images/Numeric_Prompt.PNG)
- Special Characters Prompt
![Initial_WebPage](/Develop/images/SpecialCharacters_Prompt.PNG)
- Generated Password
![Initial_WebPage](/Develop/images/Password_Generated.PNG)
